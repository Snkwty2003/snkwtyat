# منصة شنكوتيات

منصة احترافية لبيع القوالب الجاهزة وتقديم خدمات التصميم.

## المميزات

### للمستخدمين
- نظام تسجيل دخول وتسجيل حساب جديد
- لوحة تحكم شخصية للمستخدم
- عرض القوالب الجاهزة (Landing Page, Store)
- متجر إلكتروني متكامل مع سلة تسوق
- نظام البحث والفلترة
- نموذج طلب تصميم مخصص
- تصميم متجاوب لجميع الأجهزة
- استعادة كلمة المرور

### للمنصة
- نظام إدارة الطلبات
- نظام الدفع الإلكتروني
- لوحة تحكم للإدارة
- المزيد من القوالب المتنوعة

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage لحفظ البيانات

## البنية

```
.
├── index.html              # الصفحة الرئيسية
├── template-landing.html   # قالب Landing Page
├── template-store.html     # قالب المتجر
├── checkout.html           # صفحة الدفع
├── banner/                 # مجلد الصور
└── README.md               # ملف التوثيق
```
