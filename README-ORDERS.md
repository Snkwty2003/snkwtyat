# ملفات إدارة الطلبات النظيفة

تم إنشاء الملفات التالية لإدارة الطلبات بشكل نظيف ومحسّن:

## الملفات الجديدة:

1. **admin-orders-v2.html**
   - ملف HTML نظيف بالكامل
   - يحتوي على واجهة مستخدم حديثة ومتجاوبة
   - يدعم اللغة العربية (RTL)
   - يحتوي على فلاتر للبحث والتصفية

2. **admin-orders-v2.js**
   - ملف JavaScript نظيف بالكامل
   - يستخدم Firebase Firestore لجلب البيانات
   - يدعم التحديثات الفورية (Real-time updates)
   - يحتوي على معالجة أخطاء شاملة

3. **admin-orders-final.html**
   - نسخة بديلة من ملف HTML
   - مرتبط بملف admin-orders-final.js

4. **admin-orders-final.js**
   - نسخة بديلة من ملف JavaScript
   - يحتوي على نفس الوظائف الأساسية

## الميزات:

- عرض قائمة الطلبات مع التصفح (Pagination)
- تصفية الطلبات حسب الحالة
- البحث برقم الطلب أو اسم العميل
- تصفية الطلبات حسب التاريخ
- عرض تفاصيل الطلب
- تعديل حالة الطلب
- حذف الطلب
- تحديثات فورية للبيانات

## كيفية الاستخدام:

1. افتح ملف `admin-orders-v2.html` في المتصفح
2. سيتم تحميل الطلبات تلقائياً من Firebase
3. استخدم الفلاتر للبحث والتصفية
4. استخدم الأزرار لعرض وتعديل وحذف الطلبات

## المتطلبات:

- ملف `firebase-config.js` يجب أن يكون موجوداً في نفس المجلد
- يجب أن يكون لديك صلاحيات الوصول إلى قاعدة بيانات Firebase
